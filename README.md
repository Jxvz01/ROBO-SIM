# ROBO-SIM

High-fidelity 3D robotics simulator specialized for high-speed RC cars, Robo Sumo, and Robo Soccer.

## Core Tech Stack
- **Frontend:** React + Tailwind CSS (v4)
- **3D Engine:** Three.js with `@react-three/fiber` and `@react-three/drei`
- **Physics Engine:** Rapier.js (120Hz precision)
- **State Management:** Zustand

## Features
- **Assembly Manager:** Component-based bot construction with snap points.
- **Physical Realism:** Precise mass, torque, and friction calculations.
- **Environment Presets:** Sumo Ring, Soccer Pitch, and Speed Track.
